# E-PHARMACY

This application is for choosing medicines. You can buy or find the medicine you
need online the pharmacy closest to you.

---

### The following technologies were used in the development of this project:

1. Node JS
2. jsonwebtoken
3. mongoose
4. express
5. bcryptjs
6. joi
