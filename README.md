# E-PHARMACY

Це бекенд до веб-додатку для вибору ліків. Ви онлайн придбати потрібні вам ліки чи знайти
найближчу до вас аптеку.

---

### У розробці цього проекту використано такі технології:

1. Node JS
2. jsonwebtoken
3. mongoose
4. express
5. bcryptjs
6. joi
